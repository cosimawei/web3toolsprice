# 快速安装指南

## 第一步：生成图标

1. 双击打开 `generate-icons.html` 文件（用浏览器打开）
2. 你会看到三个不同尺寸的图标预览
3. 依次点击三个"下载"按钮：
   - 下载 16x16 图标
   - 下载 48x48 图标
   - 下载 128x128 图标
4. 将下载的三个 PNG 文件移动到 `icons` 文件夹中

## 第二步：安装Chrome扩展

1. 打开 Chrome 浏览器
2. 在地址栏输入：`chrome://extensions/`
3. 打开右上角的"开发者模式"开关
4. 点击左上角的"加载已解压的扩展程序"
5. 选择这个文件夹（web3文件夹）
6. 完成！

## 第三步：使用插件

1. 点击浏览器工具栏的扩展图标
2. 找到"多币种价格追踪器"并点击
3. 插件弹出窗口会显示实时价格和图表
4. 点击不同的币种按钮切换查看

## 支持的币种

- BTC (比特币)
- ETH (以太坊)
- SOL (Solana)
- BNB (币安币)
- XRP (瑞波币)
- ADA (艾达币)

## 功能特点

- **实时价格更新**: 后台每30秒自动更新BTC价格
- **图标Badge显示**: 插件图标上滚动显示BTC价格，无需打开即可查看
- **Chart.js图表**: 轻量级图表，加载快速
- **24小时价格统计**: 完整的涨跌数据
- **多时间周期切换**: 1m, 5m, 15m, 1h, 4h, 1D
- **美观的界面设计**: 渐变色背景，暗色主题
