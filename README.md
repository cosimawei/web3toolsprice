# 多资产价格追踪器 Chrome插件

一个功能强大的Chrome浏览器扩展，支持实时追踪虚拟币、Alpha代币、MEME币、股票、贵金属等多种资产价格，内嵌TradingView K线图。

## 功能特性

### 多资产类型支持
- **虚拟币**: BTC、ETH、SOL、BNB、XRP、ADA等主流币种
- **Alpha代币**: Binance Alpha平台代币，实时价格追踪
- **MEME币**: 支持BSC、ETH、Solana、Base链上的MEME代币
- **股票**: A股、港股、美股指数和个股
- **贵金属**: 黄金、白银国际价格及人民币换算

### 多交易所数据源
- Binance (WebSocket实时推送)
- OKX (WebSocket实时推送)
- Bitget (WebSocket实时推送)
- MEXC (轮询更新)

### K线图功能
- 虚拟币/股票/贵金属：TradingView专业K线图
- Alpha/MEME代币：DexScreener嵌入式K线图
- 汇率：点击汇率显示可查看USD/CNH K线

### 图标Badge滚动显示
- 插件图标上滚动显示BTC完整价格
- 绿色表示上涨，红色表示下跌
- 无需打开插件即可查看实时价格

### 其他特性
- **实时汇率**: 顶部显示美元兑离岸人民币汇率
- **拖拽排序**: 自由调整资产卡片顺序
- **云端同步**: 支持配置云端备份与恢复
- **自定义添加**: 可添加自定义代币、股票
- **页签管理**: 可隐藏不需要的资产类别

## 安装方法

### Chrome应用商店安装（推荐）
直接在Chrome应用商店搜索"多资产价格追踪器"安装

### 开发者模式安装
1. 下载本项目代码
2. 打开Chrome浏览器，访问 `chrome://extensions/`
3. 开启右上角"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择项目文件夹

## 使用说明

### 切换资产类别
点击顶部页签切换不同资产类型：
- 🪙 虚拟币 - 主流加密货币
- 🅰️ Alpha - Binance Alpha代币
- 🐸 MEME - 链上MEME代币
- 📈 股市 - A股/港股/美股
- 🥇 贵金属 - 黄金/白银

### 查看K线图
点击任意资产卡片，下方会展开对应的K线图面板

### 查看汇率K线
点击顶部的汇率显示区域，可查看USD/CNH离岸人民币K线

### 添加自定义资产
点击右上角⚙️设置按钮，进入设置页面添加：
- 虚拟币：选择交易所，输入交易对
- Alpha代币：输入代币名称
- MEME代币：选择网络，输入合约地址
- 股票：选择市场，输入股票代码

### 拖拽排序
长按并拖动资产卡片可调整显示顺序

## 数据源

| 资产类型 | 数据源 | 更新方式 |
|---------|--------|---------|
| 虚拟币 | Binance/OKX/Bitget | WebSocket实时 |
| MEXC虚拟币 | MEXC API | 5秒轮询 |
| Alpha代币 | Binance Alpha API | 10秒轮询 |
| MEME代币 | GeckoTerminal API | 15秒轮询 |
| 股票 | 腾讯股票API | 10秒轮询 |
| 贵金属 | Binance + 阿里云API | 120秒轮询 |
| 汇率 | 新浪财经API | 5分钟更新 |

## 注意事项

1. 需要网络连接才能获取实时数据
2. MEME代币需要正确的合约地址（注意大小写）
3. 贵金属中国金价需配置阿里云API AppCode
4. 部分功能可能需要科学上网
